## 📋 使用指南

### 快速啟動
```bash
cd /Users/wutsohao/salary_calculator
npm run dev
```

訪問：http://localhost:3000

### 切換模式
- **單一計算**：點擊頂部「単一計算」按鈕
- **批量計算**：點擊頂部「批量計算」按鈕

### API 調用
```bash
# 單一計算
curl -X POST http://localhost:3001/api/calculate \
  -H "Content-Type: application/json" \
  -d '{"salaryType":"monthly","baseSalary":300000,...}'

# 批量計算
curl -X POST http://localhost:3001/api/calculate/batch \
  -H "Content-Type: application/json" \
  -d '{"employees":[...]}'
```
