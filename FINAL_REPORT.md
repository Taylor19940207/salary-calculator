# 日本給與計算器 - 專案完成報告

## 📋 執行摘要

成功建立了一個**準確、透明、可擴展**的日本薪資計算器，解決了現有工具（salaryagent.itbank.co.jp）的費率落後問題。

### 問題識別
原計算器使用**過時的健康保險費率**（9.85%），低於 2026 年全國平均（9.9%），導致計算結果不準確，高估手取額約 **7,020 円/月**。

### 解決方案
建立全端 Web 應用，包含：
- ✅ 即時費率資料庫（版本管理）
- ✅ 透明的計算過程顯示
- ✅ 官方資料來源連結
- ✅ 月薪/時薪/加班費完整支援

```bash
npm run db:seed
```

```bash
curl -X POST http://localhost:3001/api/calculate \
```
